# Personal Finance Dashboard

## Overview
An interactive Personal Finance Dashboard created using Excel and Power BI to analyze income, expenses, savings, and spending behavior.

## Tools Used
- Microsoft Excel
- Power BI
- DAX

## Features
- Income Analysis
- Expense Tracking
- Savings Monitoring
- Category-wise Expense Breakdown
- Monthly Financial Trends
